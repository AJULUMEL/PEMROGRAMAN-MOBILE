# streambuilder_dandi_azrul_syahputra

Soal 12

Langkah 3 - stream.dart
Membuat class NumberStream dengan method getNumbers() yang menghasilkan Stream berisi angka random (0-9) yang diemit setiap 1 detik menggunakan Stream.periodic().

Langkah 7 - build() method
Menggunakan StreamBuilder untuk:

Listen ke numberStream secara otomatis

Handle error: jika ada error, tampilkan print('Error!')

Tampilkan data: jika ada data, tampilkan angka dengan font size 96 di tengah layar

Handle no data: jika belum ada data, tampilkan SizedBox kosong
